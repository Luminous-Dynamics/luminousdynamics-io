# Luminous Dynamics Platform

Landing page for Terra Atlas - the global energy investment platform democratizing access to renewable energy projects.

**Live site**: [luminousdynamics.io](https://luminousdynamics.io)

## About

This site serves as the entry point for Terra Atlas, featuring:
- Platform overview and mission
- Investment opportunities
- Community ownership model (Luminous Chimera Model)
- Access to the Terra Atlas application

## Local Development

```bash
# Serve locally
python -m http.server 8000
# or
npx serve .
```

Visit `http://localhost:8000`
