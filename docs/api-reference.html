    API Reference

    Complete reference documentation for all Luminous Dynamics APIs including Terra Atlas, Luminous Nix, and Sacred Bridge. Detailed endpoint specifications, parameters, and examples.

    📅 Last updated: November 2025 ⏱️ 15 minute read

    Base URL

    All API requests should be made to:

    https://api.luminousdynamics.io/v1

    Authentication

    All requests require an API key in the Authorization header:

    Authorization: Bearer YOUR_API_KEY

    Terra Atlas API

    Renewable energy site discovery and analysis.

    List Sites

    Retrieve a paginated list of renewable energy sites.

    GET /v1/sites

    Query Parameters

    Parameter Type Required Description
    page integer Optional Page number (default: 1)
    limit integer Optional Results per page (default: 20, max: 100)
    type string Optional Filter by type: solar, wind, hydro
    minCapacity number Optional Minimum capacity in kW

    Example Request

    const response = await fetch('https://api.luminousdynamics.io/v1/sites?type=solar&minCapacity=1000', {
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  }
});

const data = await response.json();

    Example Response

    {
  "sites": [
    {
      "id": "site_abc123",
      "name": "Desert Solar Farm",
      "type": "solar",
      "capacity": 5000,
      "location": {
        "lat": 34.0522,
        "lng": -118.2437,
        "address": "Los Angeles, CA"
      },
      "status": "operational",
      "efficiency": 0.87,
      "createdAt": "2025-01-15T10:00:00Z"
    }
  ],
  "total": 150,
  "page": 1,
  "pages": 8,
  "limit": 20
}

    Create Site

    Create a new renewable energy site.

    POST /v1/sites

    Request Body

    Field Type Required Description
    name string Required Site name
    type string Required One of: solar, wind, hydro
    capacity number Required Capacity in kW
    location object Required Location coordinates and address

    Luminous Nix API

    Neural intelligence and consciousness modeling.

    Analyze Consciousness

    Analyze consciousness patterns and neural signatures.

    POST /v1/analyze

    Request Body

    Field Type Required Description
    data array Required Neural data points
    algorithm string Optional Analysis algorithm (default: quantum)
    depth integer Optional Analysis depth level (1-10)

    Example Request

    const response = await fetch('https://api.luminousdynamics.io/v1/analyze', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    data: [0.45, 0.78, 0.92, 0.31],
    algorithm: 'quantum',
    depth: 7
  })
});

const analysis = await response.json();

    Example Response

    {
  "id": "analysis_xyz789",
  "patterns": [
    {
      "type": "coherence",
      "strength": 0.87,
      "confidence": 0.95
    }
  ],
  "consciousness_index": 0.92,
  "insights": [
    "High coherence detected",
    "Quantum resonance patterns present"
  ],
  "processedAt": "2025-11-15T14:30:00Z"
}

    Sacred Bridge

    Consciousness integration and transformation.

    Integrate

    Integrate consciousness patterns for transformation.

    POST /v1/integrate

    Request Body

    Field Type Required Description
    patterns array Required Consciousness patterns to integrate
    mode string Optional Integration mode: gentle, balanced, deep

    Error Codes

    The API uses standard HTTP status codes:

    Code Status Description
    200 OK Request succeeded
    201 Created Resource created successfully
    400 Bad Request Invalid request parameters
    401 Unauthorized Invalid or missing API key
    404 Not Found Resource not found
    429 Too Many Requests Rate limit exceeded
    500 Server Error Internal server error

    Error Response Format

    {
  "error": {
    "code": "validation_error",
    "message": "Invalid parameter: type must be one of solar, wind, hydro",
    "details": {
      "field": "type",
      "value": "invalid"
    }
  }
}

    Rate Limiting

    API requests are rate-limited per API key:

    • Free tier: 1,000 requests/hour
    • Pro tier: 10,000 requests/hour
    • Enterprise: Custom limits

    Rate limit information is included in response headers:

    X-RateLimit-Limit: 10000
X-RateLimit-Remaining: 9987
X-RateLimit-Reset: 1699564800
